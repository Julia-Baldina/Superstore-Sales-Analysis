# Superstore-Sales-Analysis
Анализ данных продаж Superstore
# Описание проекта
Этот проект представляет собой анализ датасета продаж Superstore (с Kaggle) с целью выявления ключевых трендов в продажах. Включает очистку данных, разведочный анализ данных (EDA), визуализации, итоговый анализ. Проект демонстрирует навыки работы с Python (Pandas, Matplotlib, Seaborn).

Датасет: Sample - Superstore.csv (с Kaggle, файл можно скачать в репрезитории)

Инсайты и визуализации

Очистка данных: Удаление дубликатов, обработка пропусков, приведение к единому виду

EDA: Анализ продаж по категориям, регионам, времени и т.д. Графики в Matplotlib/Seaborn.

Примеры графиков: Тренды прибыли, распределение по штатам и т.д.
